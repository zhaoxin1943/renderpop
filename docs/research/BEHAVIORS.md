# Creen My Assets Behavior Notes

## Image / Video tabs

- Trigger: click the corresponding toolbar button.
- Active state: dark filled pill (`rgb(37, 38, 48)`) inside a muted `rgb(32, 33, 42)` segmented container.
- Inactive state: transparent against the container with muted copy.
- Result: the date-grouped media grid changes to the selected media kind. The inspected account had no video results.

## Asset cards

- Default: `160px` high on small screens and `190px` from the `sm` breakpoint; width is derived from the source aspect ratio.
- Hover: a `2px` translucent white ring appears.
- Model badge remains visible at the top-right. It uses a semi-transparent black background, blur, 12px icon, and 12px model name.

## Create New

- Trigger: click.
- Expected behavior: enter the creation workflow.
